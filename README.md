# 川藏南线实时路况驾驶舱

汇总 G318 川藏南线沿途公开天气、官方路网通告和数据源状态。页面不会把“没有抓到通告”解释为“道路畅通”。

## 本地查看

可以直接双击 `index.html` 查看当前快照。若要测试自动刷新，请在项目目录运行：

```bash
python3 -m http.server 8000
```

然后访问 `http://localhost:8000`。

## 更新数据

```bash
python3 scripts/update_data.py
```

更新脚本使用：

- Open-Meteo：13 个沿线城镇坐标的当前天气和当日预测；
- 西藏自治区交通运输厅：最新“全区国省公路路网运行情况”通告；
- 四川省交通运输厅：检测公开路况页状态。其动态接口限制外部 IP，因此看板会明确标记为“受限”。

## 自动更新

项目已包含 GitHub Actions，每 30 分钟更新一次 `data/dashboard.json` 和离线快照。部署到 GitHub Pages 后，网页每 5 分钟重新读取数据文件。

## 重要限制

- 城镇天气不代表垭口实况；
- 政府网页的发布时间可能滞后于现场；
- 微信公众号、交警临时通知和导航事件并不一定提供可公开自动读取的接口；
- 出发前必须向 `12328` 或当地 `122` 核实具体路段及放行时间。
